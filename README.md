可视化体感控制系统 - 使用指南

🚀 快速开始
运行环境：python 3.10（Windows-64bit）
安装依赖：pip install -r requirements.txt
运行程序main.py

操作流程

启动预览: 程序启动后会自动显示主界面，点击左侧"摄像头预览"。
开始控制: 点击左下角的 "开始控制" 按钮，摄像头将启动并开始检测手势。
测试手势: 尝试对着摄像头做以下手势：
✊ 握拳 (Fist) -> 默认触发 Space
🖐 张开手掌 (Open Palm) -> 默认触发 Esc
☝ 竖起食指 (Point Up) -> 默认触发 鼠标左键点击
✌ 胜利手势 (Victory) -> 默认触发 V
👍 竖起大拇指 (Thumbs Up) -> 默认触发 Enter
✨ 自定义手势 (高级功能)
想要控制更多功能？您可以录制属于自己的手势！

新建手势:

在配置界面点击绿色的 "+ 新建手势" 按钮。
输入手势名称（例如 "比心" 或 "OK"）。
将手放入摄像头画面中，摆好您想要的手势。
点击 "开始捕获"，系统会倒计时3秒。
保持手势不动，直到捕获完成。
确认画面中的骨架是否准确，然后点击 "保存手势"。
确保摄像头已连接并正常工作。
界面采用 Google Material Design 3 风格，支持深色模式。
配置动作:

保存后，新手势会出现在列表中。
点击 "编辑" 为其分配键盘或鼠标动作。
使用技巧:

自定义手势的识别优先级高于预定义手势。
录制时请确保光线充足，手势清晰。
尽量录制与现有手势区别较大的动作，以防误触。
⚙️ 手势配置
想要修改手势对应的动作？

点击左侧导航栏的 "手势配置"。
在列表中找到您想修改的手势，点击 "编辑"。
在弹出的对话框中：
动作类型: 选择 Key (单键), Key Combo (组合键), 或 Mouse (鼠标)。
动作值: 输入对应的键值（如 ctrl+c, left_click）。
点击 "保存"，配置即时生效。
